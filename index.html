Pulse Point

Overview:
Pulse Point is an AI-powered healthcare platform that connects patients with doctors. It offers an intuitive interface for booking appointments and includes a smart chat assistant that recommends doctors based on user-reported symptoms.

Doctor Integration:
Verified doctors can register on the platform, manage their availability, and view scheduled appointments through a secure, role-specific dashboard.

User Experience:
Patients get a dedicated dashboard to manage their appointments and chat interactions. Both users and doctors receive email notifications immediately after a booking is made, ensuring timely communication and confirmation.

  • Real-time doctor appointment booking
  • Symptom-based chat assistant to suggest relevant doctors
  • Firebase Authentication with role-based access
  • Firestore & Realtime Database integration for managing data
  • Email notifications sent to both patients and doctors upon booking
  • Separate dashboards for doctors and patients
  • Responsive design for web and mobile access

Human Action Recognition

Overview:
A deep learning-based video classification system that recognizes human actions by analyzing both spatial and temporal information. The project focuses on interpreting real-time activities from videos using advanced sequential models.

Model Architecture:
Utilizes two state-of-the-art architectures — ConvLSTM and LRCN. ConvLSTM integrates convolutional filters into LSTM units for spatiotemporal learning, while LRCN combines CNNs for spatial feature extraction with LSTMs for temporal understanding.

Dataset:
Trained and evaluated on the UCF50 dataset, which consists of 50 different human action classes like walking, biking, clapping, diving, etc.

  • ConvLSTM for spatiotemporal modeling
  • LRCN: CNN + LSTM hybrid for activity recognition
  • UCF50 dataset with 50 labeled action categories
  • High accuracy on real-time video classification
  • Applications: surveillance, smart fitness, gesture recognition
